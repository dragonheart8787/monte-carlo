# 蒙地卡羅通訊效能評估系統 — 完整報告

---

## 一、摘要

本報告說明以**蒙地卡羅方法**為核心的數位通訊鏈路效能評估系統，涵蓋系統架構、方法論、實驗結果與使用方式。系統支援多種調變（BPSK、QPSK、8-PSK、16-QAM）、通道模型（AWGN、Rayleigh、Rician）、擾動與自適應抽樣，並具備理論驗證、實驗管理與 packet-level 分析能力。

**關鍵成果**：
- 模擬 BER 與理論值吻合（如 BPSK/AWGN @ 6dB：理論 2.39e-03 vs 模擬 2.45e-03）
- 支援 Wilson score 信賴區間與 CI-based 自適應停止
- 提供 Benchmark Suite、Packet Stress 與 Dashboard 產出

---

## 二、研究目的

1. **效能評估**：在不同 SNR、調變與通道條件下，估計 BER、SER、PER
2. **理論驗證**：以 closed-form 理論公式驗證模擬結果
3. **收斂分析**：觀察蒙地卡羅估計隨樣本數的收斂行為
4. **比較分析**：多調變、多通道情境的效能比較
5. **Packet-level 分析**：Burst noise、Block fading 對 PER 的影響

---

## 三、方法論

### 3.1 蒙地卡羅模擬流程

```
DataGen → Modulator → Channel (Fading + Impairment + Noise) → Receiver → Performance Eval
```

1. **資料產生**：隨機位元流或封包模式
2. **調變**：BPSK、QPSK、8-PSK、16-QAM
3. **通道**：Fading（AWGN、Rayleigh、Rician）+ Impairment（phase/freq offset）+ Noise（AWGN、Burst）
4. **接收**：Compensator → Detector → Demapper
5. **效能評估**：BER、SER、PER、Wilson score 信賴區間

### 3.2 統計方法

- **信賴區間**：Wilson score interval（適用於二項比例，小樣本與極端比例時較穩定）
- **自適應抽樣**：
  - **Theory-guided**：`run_adaptive(use_theory=True)`，依目標相對誤差分配樣本
  - **Empirical**：`run_adaptive_empirical()`，依 standard error 目標分配

### 3.3 理論支援

| 調變 | AWGN | Rayleigh | Rician |
|------|------|----------|--------|
| BPSK | ✓ closed-form | ✓ closed-form | ✓ 近似 |
| QPSK | ✓ closed-form | ✓ closed-form | ✗ |
| 8-PSK | ✓ 近似 | ✗ | ✗ |
| 16-QAM | ✓ 近似 | ✗ | ✗ |

## 五、實驗結果

### 5.1 基礎版：BPSK + AWGN，BER vs SNR

**配置**：
- 調變：BPSK
- 通道：AWGN
- SNR 範圍：0 ~ 12 dB，步長 2 dB
- 每模擬位元數：5000
- 每 SNR 試驗數：50

**結果**：

| SNR (dB) | BER (mean) | CI low | CI high |
|----------|------------|--------|---------|
| 0.0 | 7.87e-02 | 3.08e-02 | 1.87e-01 |
| 2.0 | 3.80e-02 | 1.02e-02 | 1.32e-01 |
| 4.0 | 1.24e-02 | 1.53e-03 | 9.28e-02 |
| 6.0 | 2.24e-03 | 6.15e-05 | 7.54e-02 |
| 8.0 | 2.36e-04 | 7.21e-07 | 7.18e-02 |
| 10.0 | 0.00e+00 | 6.94e-18 | 7.13e-02 |
| 12.0 | 0.00e+00 | 6.94e-18 | 7.13e-02 |

**結論**：BER 隨 SNR 增加而下降，與理論相符。高 SNR 時 BER 趨近 0，信賴區間仍可計算。

---

### 5.2 完整版：多調變與通道比較

**情境**：
1. BPSK + AWGN
2. QPSK + AWGN
3. BPSK + Rayleigh

**配置**：SNR 0~16 dB，每模擬 3000 bits，每 SNR 30 trials。

**產出**：`comparison.png` — 三條 BER vs SNR 曲線比較。Rayleigh 較 AWGN 衰減明顯，QPSK 較 BPSK 需更高 SNR。

---

### 5.3 研究版：收斂監控

**配置**：BPSK + AWGN，單點 SNR = 6 dB，200 trials。

**理論 BER @ 6dB**：2.388291e-03  
**最終模擬 BER**：2.450000e-03
**相對誤差**：約 2.6%

**結論**：模擬值與理論值吻合良好，驗證蒙地卡羅收斂行為。

---

### 5.4 Packet Stress 結論

（執行 `python run_packet_stress.py` 產出）

1. **Burst noise**：錯誤集中在少數封包，PER/BER 比上升
2. **Block fading**：同 block 內錯誤相關，PER 較 fast fading 差
3. **相同 BER**：不同錯誤分布導致不同封包成功率

## 七、使用方式

### 7.1 互動選單

```bash
pip install -r requirements.txt
python main.py
```

選項：
- 1：基礎版 (BPSK + AWGN)
- 2：完整版 (多調變/通道比較)
- 3：研究版 (收斂監控)
- 4：全部執行

### 7.2 程式化使用

```python
from mc_comm_system import SimulationConfig, MonteCarloEngine, Visualizer

config = SimulationConfig(
    modulation="BPSK",
    channel="AWGN",
    snr_db_range=(0, 12),
    bits_per_simulation=5000,
    num_trials=50,
)
engine = MonteCarloEngine(config)
results = engine.run()
Visualizer().plot_ber_vs_snr(results)
```

### 7.3 其他腳本

| 腳本 | 用途 |
|------|------|
| `run_benchmark.py` | 基準實驗回歸 |
| `run_packet_stress.py` | Packet-level 壓力故事線 |

## 八、限制與假設

（詳見 [docs/LIMITATIONS.md](LIMITATIONS.md)）

1. **Uncoded link only**：僅無編碼鏈路
2. **Coherent detection**：假設載波同步
3. **理論支援**：僅部分調變/通道有 closed-form
4. **CI stopping**：BER≈0 時極端區間不適用
5. **Impairment pipeline**：順序固定
